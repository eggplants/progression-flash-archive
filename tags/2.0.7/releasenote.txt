


Progression Framework 2.0.7



□概要
――――――――――――――――――――――――――――――――――――――――
EasyCasting 機能でのアルファ効果削除に対応した FadeMovie コンポーネントの追加、
プロジェクトアップデーターの追加、その他細かいバグフィックスを主体としたアップデ
ートです。



□リリース日
――――――――――――――――――――――――――――――――――――――――
2008/03/12



□アップデート内容
――――――――――――――――――――――――――――――――――――――――
・FadeMovie クラスおよび FadeMovie コンポーネントを追加しました。
　※これは以前、Progression Effects としてコンセプトに掲載されていた機能の実装版
　です。

・プロジェクトパネルに既存プロジェクトのアップデート機能を追加しました。
　ただし、コンポーネントアップデート時に自動的に表示される「入れ替え確認ダイアロ
　グ」で「入れ替える」を選択する操作は手動で行う必要があります。 

・Progression クラスのコンストラクタ引数である rootClass を String または Class
　で指定できるように修正しました。

・Progression クラスのコンストラクタで特定の状況下で正しくルートシーンが作成され
　ない問題を修正しました。

・GotoCommand コンポーネントのシーンパスでフォーマットエラーが出ていた問題を修正
　しました。

・ProgressionLoader コンポーネントおよび EasyCastingLoader コンポーネントで必要
　以上のデバッグ出力をしないように修正しました。

・プロジェクト種類の「標準 XML」を削除しました。 



□既知の問題
――――――――――――――――――――――――――――――――――――――――
・Windows Vista 及び Macintosh で使用した場合、プロジェクト作成でエラーが発生し
　て正しく作成できない場合があります。

・Windows 版の Opera 9.25 でページ移動を繰り返しているとフリーズする場合があるよ
　うです。

・プロジェクトパネルの種類の項目に「スライドショー」が存在し、選択してプロジェク
　トを作成すると JavaScript エラーが発生します。スライドシュー機能はまだ実装され
　ていないので、項目を選択しないようお願い致します。
