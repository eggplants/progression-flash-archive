


Progression Framework 2.0.15



□概要
――――――――――――――――――――――――――――――――――――――――
2.1.0 に向けての事前準備としてプロジェクト情報ファイル（ppinfo）の実装と、細かい
バグフィックスを主体としたアップデートです。



□リリース日
――――――――――――――――――――――――――――――――――――――――
2008/03/30



□アップデート内容
――――――――――――――――――――――――――――――――――――――――
・プロジェクトパネルにクラスパスを設定する項目が追加されます。
　この設定はプロジェクトの種類が「クラス」の場合のみ有効となります。

・クラスプロジェクトで作成されるファイルに Preloader クラスを追加しました。
　このクラスは preloader.fla に関連付けられた状態で作成されます。

・プロジェクトフォルダ内にプロジェクト情報を管理する ppinfo ファイルを作成する機
　能を追加しました。

・version 2.0.14 以前のプロジェクトフォルダをアップデートしようとした際に、プロ
　ジェクト情報ファイルの作成ダイアログを表示するように修正しました。

・JSFL を Flash IDE 上でダイレクトに使用可能な JSFL Console パネルを追加しました
　。

・ボタン系のコンポーネントで windowTarget プロパティが設定可能になりました。

・DoTweener コマンドに指定した Tweening Parameters の onComplete メソッドが実行
　されない問題を修正しました。

・ToolChip クラスはスペルが間違っていたので ToolTip クラスに名称を変更しました。

・キャストクラスの toolChip プロパティはスペルが間違っていたので toolTip プロパ
　ティに名称を変更しました。

・ProgressionLoader クラスを PRMLLoader クラスに名称を変更しました。

・ProgressionLoader コンポーネントを PRMLLoader コンポーネントに名称を変更しまし
　た。

・NextButton コンポーネント、ParentButton コンポーネント、PreviousButton コンポ
　ーネント、RootButton コンポーネントの progressionId プロパティが設定されるタイ
　ミングが、Progression インスタンスの作成前だった場合に、正しく動作しなかった問
　題を修正しました。

・RollOverButton コンポーネントの sceneId プロパティが設定されるタイミングが、
　Progression インスタンスの作成前だった場合に、正しく動作しなかった問題を修正し
　ました。

・プロジェクト書き出し時に同名のフォルダがあった場合に警告を表示するように修正し
　ました。

・プロジェクト書き出し完了時に作業用のファイルを開くかどうかのダイアログが表示さ
　れるように修正しました。

・flp ファイルと as3proj ファイルの書き出し先が src 内からプロジェクトフォルダ直
　下に変更になりました。

・コンポーネントプロジェクト及びタイムラインプロジェクトの fla ファイルで使用し
　ている読み込み用コードを外部ファイルを include する方式に変更しました。

・プロジェクトパネルの入力内容を細かくチェックして、問題があった場合にエラーを表
　示するように修正しました。

・PRMLLoader (旧 ProgressionLoader) コンポーネント及び EasyCastingLoader コンポ
　ーネントを使用している場合に、autoLock プロパティの値が強制的に false になって
　いた問題を修正しました。

・EasyCastingScene を使用している場合に、連続的なシーン移動をした際に <cast> タ
　グで指定したオブジェクトが画面から削除される問題を修正しました。

・コマンド系のインスタンスを実行した際に、コマンド名を出力するように修正しました
　。

・コマンド系のインスタンスで名前が付けられていない場合、自動的に割り振られるイン
　スタンス名を返すように修正しました。

・コマンド系のインスタンスに parent プロパティと depth プロパティを追加しました。
